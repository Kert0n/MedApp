MedApp
Курсовая работа — мобильное приложение для получения информации о лекарственных препаратах.

Описание проекта
MedApp — это система, состоящая из мобильного приложения для Android и серверной части, позволяющая пользователям вести учет лекарственных препаратов и синхронизировать общие аптечки.

Структура проекта
```
MedApp/
├── docs/                     # Документация курсовой работы (официальные документы)
│   ├── Final/                # Финальные версии документов (PDF)
│   └── WorkInProgress/       # Рабочие версии документов
├── src/                      # Исходный код
│   ├── MedAppServer/         # Серверная часть (Spring Boot)
│   ├── AndroidApp/           # Мобильное приложение (Android)
│   └── scrapper/             # Скраппер данных о препаратах (Python)
└── README.md
```
Описание компонентов 

MedAppServer/ — серверная часть приложения, реализованная на Spring Boot.

AndroidApp/ — мобильное приложение для платформы Android.

scrapper/ — скрипт для парсинга справочника лекарств vidal.ru 



Документация:
Документация курсовой работы находится в папке docs/:

Техническое задание (ТЗ) — описание требований к системе
Описание программы — описание архитектуры и функционала
Текст программы — исходный код и описание реализации
